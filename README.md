# 12306 抢票工具 README

## 项目简介

本项目旨在帮助用户通过自动化方式抢购中国铁路客户服务中心（12306）网站的火车票。该工具通过模拟用户行为，快速提交购票请求，提高购票成功率。

## 功能特点

- **自动登录**：支持用户自动登录12306账户。
- **快速查询**：实时查询可用的火车票信息。
- **自动抢票**：设定抢票策略，自动提交订单。
- **通知提醒**：成功抢票后，通过邮件或短信通知用户。
